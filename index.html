Always true

The standing rules
  • Sub-agents read; main session writes — every write verified by re-read.
  • Flow chart runs only after a 100% green review.
  • No questions after Step 0. Nothing skipped.
  • Frames never deleted — hidden via visible=false; reparent before removing a section.
  • Endpoints & screens never invented — unknowns = TBD; missing screens cloned.
  • One master DS per run. Only 3 pauses allowed.
